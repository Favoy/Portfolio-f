Titanic Survival Prediction

Developed a classification model to predict passenger survival using demographic and ticket data from the Titanic dataset.
Tools: Python, Pandas, Scikit-learn, Matplotlib
Approach: Applied feature engineering (e.g., title extraction), handled missing values, and optimized multiple models including Logistic Regression and Random Forest.
Outcome: Achieved 80%+ accuracy and identified key survival predictors such as gender, age, and class.

Credit Card Fraud Detection

Created a model to detect fraudulent credit card transactions within an imbalanced dataset.
Tools: Python, Pandas, Scikit-learn, SMOTE
Approach: Used oversampling (SMOTE) to handle class imbalance, trained a Random Forest classifier, and evaluated with ROC and PR AUC metrics.
Outcome: Improved recall significantly, increasing the model's ability to detect rare fraud cases accurately.
