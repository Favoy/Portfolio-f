Titanic Survival Prediction

Objective: Predict passenger survival on the Titanic using demographic and travel-related features. Tools: Python, Pandas, Scikit-learn, Matplotlib. Approach: Built and compared Logistic Regression, Random Forest Classifier and other models using cleaned Titanic dataset. Applied feature engineering (e.g., title extraction, family size), handled missing values, and tuned hyperparameters for optimal accuracy. Result: Achieved over 80% accuracy. Identified key survival predictors such as gender, ticket class, and age group.

Credit Card Fraud Detection

Objective: Detect fraudulent credit card transactions in an imbalanced dataset to support real-world financial risk mitigation. Tools: Python, Scikit-learn, Pandas, SMOTE Approach: Applied oversampling using SMOTE to balance the dataset. Trained a Random Forest Classifier and evaluated model performance using Precision-Recall AUC and ROC AUC scores. Result: Boosted recall significantly, enabling the model to detect rare fraud cases. Highlighted key features like transaction amount and time.
